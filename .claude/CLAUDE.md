# ABM Enerji Website

## Tech Stack
- Next.js 14+ (App Router), TypeScript, Tailwind CSS v4, Framer Motion

## CRITICAL RULES
- ABSOLUTELY NO PURPLE anywhere (no violet, no lavender, no magenta, no #800080)
- Color palette: Gold (#E8A317/#D4A017), Navy (#1B2A4A/#0C1929), Slate neutrals, Amber accent
- All text content is in Turkish
- Animations: subtle fade-up only, 0.5-0.7s, `once: true`
- Mobile-first responsive design
- Left-aligned text (not centered, except stats section)

## File Structure
- `src/components/` — Section components (Navbar, Hero, About, etc.)
- `src/components/ui/` — Reusable UI primitives (Button, Container, SectionHeading)
- `src/lib/` — Constants, animation variants, utilities
- `src/hooks/` — Custom React hooks
- `public/` — Static assets
